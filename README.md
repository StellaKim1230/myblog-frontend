# Stella's Blog Front

## Skill Set

- react.js: 16.13.1
- typescript: 3.9.6
- testing library: jest

## 설치 및 실행

```bash
$ git clone
$ yarn (모듈 설치)
$ yarn start (실행)
$ yarn test (테스트코드 실행)
```

## branch strategy

- 티켓번호 : SBF-1 (SBF : Stella Blog Front, 번호는 git issue 번호와 동일)
- 운영 : master
- 개발 : develop
- 기능 추가 및 유지보수 : feature/\*
- 버그 수정 : bugfix/\*
- 핫픽스 : hotfix/\*
- 릴리즈 : release/\*
